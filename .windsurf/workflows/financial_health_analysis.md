# 财务健康分析工作流

这个工作流将帮助您分析个人或家庭的财务健康状况，基于您提供的财务数据。

## 1. 准备工作

1. 确保您已经获取了最新的资产数据
2. 确保您已经获取了最新的交易数据（至少最近12个月）
3. 准备以下信息：
   - 月收入
   - 月支出（分固定支出和非必要支出）
   - 月债务还款额
   - 信用卡信息（额度和已用额度）
   - 年度收入和支出
   - 教育金准备情况（如适用）
   - 保险覆盖情况

## 2. 月度财务健康指标分析

### 2.1 流动性指标

1. 计算流动比率：
   ```
   流动比率 = 流动资产 / 流动负债
   ```
   - 健康标准：1.5-3.0
   - 如果低于1.5，建议增加流动资产或减少短期负债

2. 计算应急资金覆盖率：
   ```
   应急资金覆盖率 = 应急资金 / 月均支出
   ```
   - 健康标准：3-6个月
   - 如果不足，建议逐步增加应急资金

### 2.2 储蓄能力指标

1. 计算储蓄率：
   ```
   储蓄率 = (月收入 - 月支出) / 月收入 × 100%
   ```
   - 健康标准：≥20%
   - 如果低于20%，建议审查并优化支出结构

2. 计算债务偿还比率：
   ```
   债务偿还比率 = 月债务还款额 / 月收入 × 100%
   ```
   - 健康标准：≤35%
   - 如果超过35%，建议制定债务偿还计划

### 2.3 消费结构指标

1. 计算固定支出比：
   ```
   固定支出比 = 固定支出 / 月收入 × 100%
   ```
   - 健康标准：≤50%
   - 如果超过50%，建议审查固定支出项目

2. 计算非必要支出比：
   ```
   非必要支出比 = 非必要支出 / 月收入 × 100%
   ```
   - 健康标准：≤30%
   - 如果超过30%，建议优化消费习惯

### 2.4 债务管理指标

1. 计算信用卡使用率：
   ```
   信用卡使用率 = 信用卡已用额度 / 信用卡总额度 × 100%
   ```
   - 健康标准：≤30%
   - 如果超过30%，建议减少信用卡使用或申请提高额度

## 3. 年度财务健康指标分析

### 3.1 净资产状况

1. 计算净资产增长率：
   ```
   净资产增长率 = (年末净资产 - 年初净资产) / 年初净资产 × 100%
   ```
   - 健康标准：≥5%
   - 如果低于5%，建议优化投资策略或增加收入来源

2. 计算投资资产比：
   ```
   投资资产比 = 投资性资产 / 总资产 × 100%
   ```
   - 健康标准：≥50%
   - 如果低于50%，建议增加投资比例

### 3.2 收入多元化

1. 统计收入来源数量：
   - 健康标准：≥3个
   - 如果不足3个，建议开发新的收入来源

2. 计算被动收入比：
   ```
   被动收入比 = 被动收入 / 总收入 × 100%
   ```
   - 健康标准：≥10%
   - 如果低于10%，建议增加被动收入来源

### 3.3 债务管理

1. 计算资产负债率：
   ```
   资产负债率 = 总负债 / 总资产 × 100%
   ```
   - 健康标准：≤50%
   - 如果超过50%，建议制定减债计划

2. 计算债务收入比：
   ```
   债务收入比 = 年债务支出 / 年收入 × 100%
   ```
   - 健康标准：≤35%
   - 如果超过35%，建议优化债务结构

## 4. 特殊指标分析

### 4.1 家庭财务健康

1. 计算子女教育金准备率（如适用）：
   ```
   子女教育金准备率 = 已准备教育金 / 预计教育金需求 × 100%
   ```
   - 根据子女年龄设定目标准备率

2. 计算保险覆盖率：
   ```
   保险覆盖率 = 保险保额 / 年收入
   ```
   - 健康标准：5-10倍年收入
   - 如果不足，建议适当增加保险配置

### 4.2 投资健康

1. 评估投资组合分散度：
   - 健康标准：跨3-5个不相关资产类别
   - 如果不足，建议分散投资

2. 计算投资回报率：
   ```
   投资回报率 = (期末投资价值 - 期初投资价值) / 期初投资价值 × 100%
   ```
   - 健康标准：跑赢通胀率+3%
   - 如果未达标，建议调整投资组合

## 5. 评估结果与建议

1. 汇总各项指标得分
2. 识别需要改进的领域
3. 制定具体的改进计划
4. 设定下次评估时间（建议每季度评估一次）

## 6. 执行与跟踪

1. 实施改进计划
2. 定期（建议每月）跟踪关键指标
3. 根据实际情况调整财务计划

## 7. 专业咨询建议

对于以下情况，建议寻求专业财务顾问：
- 复杂的税务规划需求
- 大额资产配置决策
- 退休规划
- 遗产规划
- 复杂的投资产品配置
