# 年度消费分析工作流

## 1. 分类占比与趋势分析

### 数据获取
- 查询参数：
  - 按年份筛选数据
  - 按分类聚合支出金额
  - 计算各分类占比

### 分析维度
1. **年度支出总额**
   - 计算每年总支出
   - 计算同比增长率

2. **分类占比**
   - 主要支出分类：
     - 住房
     - 百货
     - 餐饮
     - 交通
     - 医疗
     - 旅游
     - 礼物
     - 其他

3. **趋势分析**
   - 各分类年度变化趋势
   - 异常值分析（如大额支出）
   - 季节性波动

## 2. 消费结构分析

### 固定支出 vs 弹性支出

#### 固定支出
- **住房**：租金/房贷
- **贷款**：车贷及利息
- **保险**：各类保险费用

#### 弹性支出
- **日常消费**：餐饮、交通
- **可选消费**：旅游、娱乐
- **一次性大额支出**：大件商品、医疗

### 必需支出 vs 非必需支出

#### 必需支出
- 住房
- 基本生活（食品、日用品）
- 医疗
- 交通（通勤）
- 教育

#### 非必需支出
- 旅游
- 娱乐
- 奢侈品
- 餐饮（超出基本需求部分）

## 4. 优化建议

### 支出优化
1. **住房支出**
   - 考虑重新协商租金
   - 评估换租可能性

2. **医疗支出**
   - 增加预防性医疗支出
   - 考虑补充商业保险

3. **弹性支出管理**
   - 设置百货和礼物预算
   - 监控餐饮支出增长

### 收入优化
1. 利用收入增长增加储蓄
2. 建立应急基金
3. 考虑多元化投资

## 5. 后续分析建议
1. 按月分析消费波动
2. 跟踪特定消费类别的变化
3. 设置预算目标并监控执行情况
4. 定期（季度/半年）进行支出审查
