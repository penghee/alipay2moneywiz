# 🚀 快速开始指南

## 第一步：基础设置（第1周）
1. **创建财务快照**
   - 记录当前所有资产（现金、存款、投资等）
   - 列出所有负债（贷款、信用卡等）
   - 计算当前净资产（资产 - 负债）

2. **设置基础账户**
   - 开设专用银行账户用于日常开支
   - 设置自动转账到储蓄账户
   - 下载并连接常用的银行/支付应用

## 第二步：建立预算（第2-3周）
1. **追踪30天支出**
   - 记录每一笔支出
   - 按类别分类（餐饮、交通、娱乐等）
   - 使用预算管理工具或电子表格

2. **制定第一个预算**
   - 设定合理的支出限额
   - 确定固定支出和可调整支出
   - 预留应急资金

## 第三步：建立应急基金（1-3个月）
1. 目标：存够3-6个月生活费的应急资金
2. 存放位置：高流动性账户（如货币基金）
3. 每月自动存款：至少收入的10%

## 第四步：基础保险配置（第2个月）
1. 优先级顺序：
   - 医疗险
   - 重疾险
   - 定期寿险（如有家庭责任）
   - 意外险

## 第五步：开始投资（第3个月起）
1. 从低风险产品开始：
   - 货币基金
   - 债券基金
   - 指数基金定投

## 📱 账单导出指南

### 支付宝账单导出步骤
1. **手机端导出**：
   - 打开支付宝App → 我的 → 账单
   - 点击右上角"..." → 开具交易流水证明
   - 选择"用于个人对账" → 选择时间范围（建议按月导出）
   - 填写接收邮箱 → 点击"发送"

2. **网页版导出**：
   - 登录支付宝官网(www.alipay.com)
   - 进入"交易记录" → 点击"下载查询结果"
   - 选择时间范围 → 下载Excel格式文件

### 微信账单导出步骤
**手机端导出**：
   - 打开微信 → 我 → 服务 → 钱包
   - 点击"客服中心" → 账单下载
   - 选择"用于个人对账" → 选择时间范围
   - 填写邮箱地址 → 确认发送

### 导入系统说明
1. 登录系统 → 右上角
2. 点击"导入" → 选择对应的账单文件
3. 系统自动识别并匹配交易类别
4. 确认导入 → 完成

### 实用小贴士
- 使用"50/30/20"法则开始：
  - 50% 必要支出
  - 30% 非必要支出
  - 20% 储蓄与投资
- 每月固定时间进行财务回顾
- 设置财务目标（短期、中期、长期）

### 常见问题
Q: 应该先还债还是先存钱？
A: 建议先存够1个月应急资金，然后集中还高息债务，最后再增加应急资金到3-6个月。

Q: 每月应该存多少钱？
A: 建议至少收入的20%，如果暂时达不到，可以从10%开始逐步提高。

Q: 系统支持哪些文件格式导入？
A: 系统支持Excel/CSV格式文件，建议使用支付宝/微信导出的原始文件直接导入。
